## Requirements

- ProcessWire 3.0.229 or newer
- PHP 7.x or newer (preferably PHP 8.x)
- MySQL 5.7 or newer 

## Installation

1. Extract the module files to /site/modules/FieldtypeCustom/
2. In your admin go to Modules > Refresh.
3. Click "Install" for FieldtypeCustom.
4. Go to Setup > Fields > Add New.
5. Choose ProFields > Custom. 
6. Follow the on-screen instructions for the rest.

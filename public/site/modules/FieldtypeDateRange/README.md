# ProcessWire ProFields: Date Range Fieldtype and Inputfield

**This is part of the [ProFields](https://processwire.com/store/pro-fields/) 
package of modules. Please do not distribute.**

Date Range fields enable input, storage and querying of two dates ("date from" 
and "date to"). This is useful for anything that requires a starting and 
ending date. A single-date selection mode is also available. This module can 
be used for fields in ProcessWire, FormBuilder, or any other Inputfield-based 
form. 

More about Date Range fields: 
<https://processwire.com/blog/posts/date-range-fields/>

## Installation

1. Copy all the files in this directory to `/site/modules/FieldtypeDateRange/`
2. In your admin, go to Modules > Refresh.
3. Click the "Install" button next to FieldtypeDateRange. 

## Documentation and API

Please see the following link for documentation and API details:  
<https://processwire.com/blog/posts/date-range-fields/>
